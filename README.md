
# Mental Health & Academic Productivity Analysis

This repository contains a data analysis project exploring the relationship between students' mental health and their academic productivity. The analysis uses Python and includes statistical modeling and data visualization.

## 📌 Project Overview
The goal of this project is to investigate whether there is a quantifiable relationship between mental health scores (`mental_health_score`) and productivity scores (`productivity_score`) of students. We use a dataset of 5000 students to perform correlation analysis, build a linear regression model, and visualize the findings.

## 📊 Key Findings
- **Correlation:** A Pearson correlation coefficient of **0.63** was found, indicating a moderate to strong positive relationship.
- **Regression Model:** A simple linear regression was performed, yielding the equation: `Productivity = 16.87 + 3.70 * Mental Health`
- **Model Strength:** The model explains **39.8%** of the variance in productivity (R-squared = 0.398).
- **Statistical Significance:** The relationship is statistically significant (P-value < 0.001).

## 🛠️ Tools & Libraries Used
- **Language:** Python
- **Data Manipulation:** Pandas
- **Data Visualization:** Matplotlib, Seaborn
- **Statistical Modeling:** Statsmodels
- **Environment:** Jupyter Notebook

## 📁 Repository Contents

## 🚀 How to Run
1.  Ensure you have Python installed with the required libraries (`pandas`, `matplotlib`, `seaborn`, `statsmodels`).
